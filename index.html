HelMail Privacy Policy

This privacy policy describes how the HelMail Chrome extension handles user data.

Data Collection and Usage

HelMail does not collect or store any personal user data. It operates solely within the user's browser to extract email addresses from the currently active webpage. The extracted email addresses are only displayed within the extension's popup window or downloaded as a text file if the user chooses to do so.

The extension does not transmit any data to external servers or third parties. All processing occurs locally within the user's browser.

Cookies and Tracking

HelMail does not use cookies or any other tracking technologies.

Changes to This Policy

As HelMail does not collect any user data, we do not anticipate frequent changes to this privacy policy. If any changes are made, we will update this page accordingly.
